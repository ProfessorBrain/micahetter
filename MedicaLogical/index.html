MedicaLogical

How to play

Match each adult patient and clinical detail to the correct appointment time.

  1. Read the clues, then choose one item in each row for every appointment.
  2. Each option can be used only once per row.
  3. Applied clues move to the bottom. A red clue means the current choices conflict.
  4. Some clues require medical knowledge to identify a diagnosis, treatment, or symptom.
  5. Difficulty changes how directly the clues reveal each relationship.
